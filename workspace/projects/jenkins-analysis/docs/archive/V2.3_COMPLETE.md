## Technical Details

### Code Refactoring

**Old approach:**
1. Query database
2. Count categories from DB
3. Generate breakdown
4. Generate table

**Problem:** DB has all entries (including duplicates), counts were inflated

**New approach (V2.3):**
1. Loop through failed jobs
2. Build table rows (with deduplication)
3. **Count categories while building rows**
4. Generate breakdown from table counts
5. Append table

**Key code:**
```javascript
// Collect row data and counts simultaneously
tableRows.push(rowString);
tableCategoryCounts[category] = (tableCategoryCounts[category] || 0) + 1;
tableErrorTypeCounts[errorType] = (tableErrorTypeCounts[errorType] || 0) + 1;

// Generate breakdown from collected counts
Object.entries(tableCategoryCounts).forEach(([cat, count]) => {
  // Render category row
});
```

## Scenarios

### TC83064 - Functional [Workstation][Report Editor] Advanced banding
1. Create report: Subcategory, Category; Cost, Cost Growth, Profit, Profit Margin
2. Enable banding
3. Verify row background colors (white, gray alternating)
4. Select banding by rows, apply color every 3
5. Change first/second banding colors (Peach, Porcelain)
6. Apply color by Row Header, select Category
7. Select banding by Column, Metrics as header
8. Apply color by Number of Columns
9. Set apply color every 1
10. Change colors (Coral, Silver)

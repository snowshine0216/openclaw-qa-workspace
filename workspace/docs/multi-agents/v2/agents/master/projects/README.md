# Projects - master

Artifacts and deliverables for the **master** agent.

## Structure

Organize files by:
- **Issue key:** `BCIN-1234/`
- **Date:** `YYYY-MM-DD/`
- **Category:** `test-reports/`, `test-plans/`, etc.

## Isolation

This folder is specific to **master**. Other agents have their own `projects/` folders.

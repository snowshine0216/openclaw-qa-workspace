# Report Editor — Percent to Total for Metrics

**Seed:** `tests/seed.spec.ts`

## TC85613_4 — Creating percent to total metrics (rows and columns)

### Steps
1. Edit report by URL (dossier ReportGridShortcutMxAttrInCols)
2. Switch to design mode
3. Verify grid cell (0,0) has text "Year"
4. Create percent to total for Cost: Over Rows
5. Verify grid shows "Percent to Total By Rows (Cost)"
6. Create percent to total for Cost: Over Columns
7. Open context menu for "Percent to Total By Rows (Cost)" → Edit...
8. Save formula metric
9. Open context menu for "Percent to Total By Columns (Cost)" → Edit...
10. Save formula metric

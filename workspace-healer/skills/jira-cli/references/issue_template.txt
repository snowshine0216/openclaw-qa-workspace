h3. Description
After logging in, the tab is restored to the first one instead of maintaining the user's last active tab.

h3. Steps to Reproduce
# Log in to the application
# Navigate to any tab other than the first one
# Perform some actions
# Log out or close the session
# Log in again

h3. Expected Result
User should be redirected to the last active tab they were using before logout.

h3. Actual Result
The application always restores to the first tab regardless of user's last active tab.

h3. Impact
* Poor user experience
* Loss of workflow continuity
* Users must manually navigate to their working tab each time

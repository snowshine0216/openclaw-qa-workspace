## Answer

MicroStrategy Workstation has replaced the legacy prompt editor with its new Dashboard Editor, which includes most features from the Library Dashboard Editor. The Library interface is now the primary interface, phasing out the Web interface. Workstation's features are increasingly aligned with those of the Library.
